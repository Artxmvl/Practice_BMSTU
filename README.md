# Проект: Нахождение минимального значения

## Описание
Проект содержит программу на C++, которая позволяет пользователю ввести массив чисел и находит минимальное значение среди них.

## Функциональность
1. Ввод количества элементов массива
2. Ввод элементов массива с клавиатуры
3. Нахождение минимального значения
4. Вывод результата
