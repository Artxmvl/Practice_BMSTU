# Проект: Нахождение максимального значения

## Описание
Проект содержит программу на C++, которая позволяет пользователю ввести массив чисел и находит максимальное значение среди них.

## Функциональность
1. Ввод количества элементов массива
2. Ввод элементов массива с клавиатуры
3. Нахождение максимального значения
4. Вывод результата
