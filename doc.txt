Метод findMax:

Описание: Метод находит максимальное значение в переданном списке целых чисел.

Параметры:
- const vector<int>& numbers - список целых чисел

Возвращаемое значение:
- int - максимальное значение в списке
- INT_MIN - если список пуст

Алгоритм:
1. Проверка на пустоту списка
2. Инициализация переменной max первым элементом
3. Поиск максимального элемента в цикле
4. Возврат результата

Пример использования:
vector<int> nums = {1, 5, 3, 9, 2};
int result = findMax(nums); // вернет 9
